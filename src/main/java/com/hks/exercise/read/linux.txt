文件目录：
  /proc：此目录的数据都在内存中，如系统核心，外部设备，网络状态，由于数据都存放于内存中，所以不占用磁盘空间
  /etc：包含很多文件.许多网络配置文件也在/etc 中.
  /sys：文件系统访问 Linux内核
  /bin：可执行二进制文件的目录，如常用的命令ls、tar、mv、cat等。
  /usr：应用程序存放目录。
      /bin：存放应用程序
      /share：存放共享数据
        /doc：系统说明文件存放目录
        /man：程序说明文件存放目录
      /lib： 存放不能直接运行的，却是许多程序运行所必需的一些函数库文件
      /local：存放软件升级包
  /var：放置系统执行过程中经常变化的文件
      /log：随时更改的日志文件
  /tmp：一般用户或正在执行的程序临时存放文件的目录,任何人都可以访问,重要数据不可放置在此目录下
  /：根目录
  ~：用户的home目录,对应地址为：/home/{userName}目录


命令：
   pwd：输出当前目录
   clear：清空终端
   du：功能该命令是显示指定文件以及下的所有文件占用系统数据块的情况，如果没有文件，默认为是当前工作目录
     -a    显示所有文件对系统数据块的使用情况
     -b    显示数据块大小时以字节为基本单位
     -c    除了显示文件对系统数据块的使用情况外还显示出文件的总和
     -s    只显示文件数据块总大小，不显示文件名
     -x    只显示当前目录，但不统计子目录
   top：相当于windows下的资源管理器，可以查看当前CPU的使用情况，各进程的状况
     -b    批处理
     -i    忽略过失
     -c    显示完整的治命令
     -s    保密模式
   free：显示所有内存利用率。包括实体内存，虚拟的交换文件内存，共享内存区段，以及系统使用的缓冲区等。
     -b    以bype为单位显示
      -k    以KB为单位显示
      -m    以MB为单位显示
      -o    不显示缓冲区
      -t    显示内存总和列
   pstack 进程id：进程跟踪
   su 用户名（不输为root）：切换用户(由root用户向普通用户切换时不需要密码，由普通用户向root用户切换时需要密码)
   password：修改密码
   ps：用于显示当前进程的状态。
   grep：找文件里符合条件的字符串。
   vim 打开文件编辑
   |：将两个命令隔开，管道符左边命令的输出就会作为管道符右边命令的输入。
   cat：文本输出命令，通常是用于观看某个文件的内容
      一次显示整个文件： cat filename
      从键盘创建一个文件 cat > filename
      将几个文件合并为一个文件  cat file1 file2 > file
      参数：
        -n  输出的文件中每行添加序列号,从1开始
        -b  和-n差不多，忽略空白行
        -v  多行空白合并成一行
   head [args] 文件名   默认输出一个文件的前10行内容
        和cat结合使用时cat 文件路径 | head -1
      -q 隐藏文件名
      -v 显示文件名
      -c<字节> 显示字节数
      -n<行数> 显示的行数，简写-1：第一行
   tail 显示文件后几行和head使用一致
   cut  文件内容查看，显示行中的指定部分，删除文件中指定字段，以行为操作
      -d：指定分隔符,默认为TAB
      -f：显示指定字段的内容；
      -complement 排除显示-f指定的
      -c 指定按操作字符
      cut -f 1 test.txt   输出文件中每行第一个TAB之前的内容
      cut -f 2,3 test.txt  输出文件中每行第2,3个TAB的内容
      cut -f 1 -d";" test.txt  分隔符不以TAB,用;分割

文件操作（支持*匹配,s*匹配所有s开头的）
    ls 列出所有文件和文件夹 file 文件名 显示文件的编码格式
    vim 文件名  ->  创建文件并打开文本编辑器
      打开后先是在命令模式，控制屏幕光标的移动，字符、字或行的删除，移动复制某区段及进入Insert mode下，或者到 last line mode。
        按「a」进入插入模式后，是从目前光标所在位置的下一个位置开始输入文字；
        按「o」进入插入模式后，是插入新的一行，从行首开始输入文字。
        退出先按Esc进入命令模式后,输入
        :wq ->  保存文件并退出
        :w  ->  保存文件不退出
        :w  ->  文件名 另存为
        :q! ->  不保存强制退出文件
    mkdir 文件夹   ->  创建文件夹
    rmdir 文件夹   ->  删除文件夹
        -p   删除后为空文件夹也一并删除
    rm 文件名  ->  删除文件
    cp 文件名 目的地  ->  拷贝文件
    mv 文件名 目的地  ->  移动文件,移动到本路径可以修改文件名
    cat 文件名 ->  读取并输出文件
    wc -l 文件名   ->  统计文件行数
    touch 创建文件
    rm 文件夹名 -rf   删除非空文件夹
    netstat -ntlp    查看端口占用


iptables
iptables -L -n  查看所有开放端口映射
iptables -A INPUT -p tcp -m tcp --dport 6066 -j ACCEPT    添加开放端口
iptables -A OUTPUT -p tcp --dport 6066 -j DROP            删除开放端口
iptables -A INPUT -s 192.168.0.1 -p tcp --dport 3306 -j ACCEPT   运行固定ip访问固定端口
service iptables save 或者 iptables-save>/etc/sysconfig/iptables 保存配置
